Hello World!

This is a vim tutorial!

To get started:
* i = begin inserting text at the current cursor position
* a = begin inserting text after the current cursor position

How to move around in vim you ask?
* H = left
* K = up
* L = right
* J = down

Some vim commands (capital letters matter here):
* Y = copies a line of text to  the buffer
* P = paste it to the cursor's current position
* dd = delete the whole line of text
* yy = copies a whole line of text
* :w = saves a file

How to quit vim:
* :wq = write (save) and quit file (and vim)
* :q! = quit and ignore changes made since last file save

Let's try them out!
